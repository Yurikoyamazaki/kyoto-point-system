
## 🚀 セットアップ手順（30分）

### ステップ1: Node.jsをインストール（5分）

#### 1-1. ダウンロード

1. ブラウザで以下にアクセス:
   ```
   https://nodejs.org/
   ```

2. 緑色のボタン「20.x.x LTS」をクリック

3. ダウンロードが完了するまで待つ

#### 1-2. インストール

1. ダウンロードした `node-v20.x.x-x64.msi` をダブルクリック

2. インストーラーが起動したら:
   - 「Next」をクリック
   - ライセンスに同意して「Next」
   - インストール先はそのまま「Next」
   - 「Install」をクリック
   - 完了まで待つ（2-3分）
   - 「Finish」をクリック

#### 1-3. 確認

1. **Windowsキー + R** を押す

2. `cmd` と入力してEnter

3. 以下を入力してEnter:
   ```cmd
   node --version
   ```

4. バージョンが表示されればOK（例: `v20.11.0`）

---

### ステップ2: プロジェクトをダウンロード（3分）

#### 方法A: ZIPファイルでダウンロード（簡単・推奨）

1. ブラウザで以下にアクセス:
   ```
   https://github.com/Yurikoyamazaki/kyoto-point-system
   ```

2. 緑色の「Code」ボタンをクリック

3. 「Download ZIP」をクリック

4. ダウンロードが完了したら、ZIPファイルを右クリック

5. 「すべて展開」を選択

6. 展開先を選択（例: `C:\`）

7. 「展開」をクリック

8. フォルダ名を確認:
   ```
   C:\kyoto-point-system-main\
   ```

#### 方法B: Git Clone（Gitインストール済みの場合）

1. コマンドプロンプトを開く

2. 以下を実行:
   ```cmd
   cd C:\
   git clone https://github.com/Yurikoyamazaki/kyoto-point-system.git
   cd kyoto-point-system
   ```

---

### ステップ3: 依存関係をインストール（10分）

#### 3-1. プロジェクトフォルダを開く

1. エクスプローラーで以下を開く:
   ```
   C:\kyoto-point-system-main\
   ```
   （ZIPでダウンロードした場合は `-main` が付きます）

2. アドレスバー（上部のパス表示部分）をクリック

3. `cmd` と入力してEnter

4. コマンドプロンプトが開く

#### 3-2. ルートの依存関係をインストール

コマンドプロンプトで以下を実行:

```cmd
npm install
```

**表示例:**
```
npm WARN deprecated ...
added 1234 packages in 2m
```

**所要時間:** 2-3分

**注意:** 警告（WARN）が表示されても問題ありません

#### 3-3. デモの依存関係をインストール

続けて以下を実行:

```cmd
cd packages\demo
npm install
```

**所要時間:** 1-2分

---

### ステップ4: デモを起動（2分）

#### 4-1. デモサーバーを起動

コマンドプロンプトで以下を実行:

```cmd
npm run dev
```

**表示例:**
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.1.100:5173/
  ➜  press h + enter to show help
```

#### 4-2. ブラウザでアクセス

1. ブラウザ（Chrome、Edge等）を開く

2. アドレスバーに以下を入力:
   ```
   http://localhost:5173
   ```

3. Enterキーを押す

4. デモページが表示される！

## 📱 スマートフォンで確認

同じWi-Fi内のスマホからもアクセスできます。

### 手順

1. **PCのIPアドレスを確認**

   コマンドプロンプトで:
   ```cmd
   ipconfig
   ```

   `IPv4 アドレス` を確認（例: 192.168.1.100）

2. **スマホのブラウザでアクセス**

   スマホのブラウザで:
   ```
   http://192.168.1.100:5173
   ```

3. **デモを確認**

   スマホで実際のアプリのように操作できます

## 🛑 停止方法

### デモを停止

1. コマンドプロンプトをアクティブにする

2. **Ctrl + C** を押す

3. 以下のメッセージが表示される:
   ```
   Terminate batch job (Y/N)?
   ```

4. `Y` を入力してEnter

5. デモが停止する

## 🔧 トラブルシューティング

### 問題1: Node.jsが認識されない

**症状:**
```
'node' は、内部コマンドまたは外部コマンド...として認識されていません
```

**解決方法:**
1. コマンドプロンプトを閉じて、もう一度開く
2. それでもダメならPCを再起動
3. Node.jsを再インストール

---

### 問題2: npm installが失敗

**症状:**
```
npm ERR! code ENOENT
npm ERR! syscall open
```

**解決方法:**

```cmd
# キャッシュをクリア
npm cache clean --force

# 再インストール
npm install
```

---

### 問題3: ポート5173が使用中

**症状:**
```
Port 5173 is in use, trying another one...
```

**解決方法A: 自動で別のポートを使用**

表示されたURLにアクセス（例: `http://localhost:5174`）

**解決方法B: 手動で別のポートを指定**

```cmd
npm run dev -- --port 3000
```

アクセス: `http://localhost:3000`

---

### 問題4: ブラウザで表示されない

**確認事項:**
1. コマンドプロンプトにエラーが表示されていないか
2. URLが正しいか（`http://localhost:5173`）
3. ブラウザのキャッシュをクリア（Ctrl + F5）

**解決方法:**

```cmd
# デモを停止
Ctrl + C

# 再起動
npm run dev
```

---

### 問題5: スマホからアクセスできない

**確認事項:**
1. PCとスマホが同じWi-Fiに接続されているか
2. PCのIPアドレスが正しいか
3. PCのファイアウォールが許可しているか

**解決方法:**

Windowsファイアウォールで許可:
1. Windowsキーを押して「ファイアウォール」と検索
2. 「Windows Defender ファイアウォール」を開く
3. 「詳細設定」をクリック
4. 「受信の規則」→「新しい規則」
5. 「ポート」を選択
6. 「TCP」、「特定のローカルポート」に `5173` を入力
7. 「接続を許可する」を選択
8. 完了
